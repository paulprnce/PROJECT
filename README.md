Project Overview

This project focuses on predicting bioethanol yield (g/L) from lignocellulosic biomass using machine learning and presenting the results through a professional, interactive web dashboard.

The system integrates:

-A trained regression model for ethanol yield prediction

-Process-aware input constraints to reflect real fermentation conditions

-An interactive Streamlit-based web interface with modern visualizations

-The goal is to support process optimization, decision-making, and educational insight in bioethanol production.
